## Project Overview: studytrack-senior-project
StudyTrack is a web-based task and progress management system designed for small student teams.
It helps students organize tasks, track deadlines, collaborate with teammates, and visualize progress.
The goal is to provide a simple, user-friendly alternative to overly complex productivity tools.
